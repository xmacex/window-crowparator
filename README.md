# Window crowparator

Window comparator for crow and norns.

Set window width in params. Also redefine truth and non-truth there to be what you wish. What is truth even? Crow will tell.

## Inputs and outputs

    → 1 window center
    → 2 input
      1 above   →
      2 inside  →
      3 outside →
      4 below   →

<kbd>E1</kbd>, <kbd>E2</kbd> and <kbd>E3</kbd> change the three parameters for window width, truth voltage and falsehood voltage.

## Requirements

- crow
- norns

## Installation

```
;install https://github.com/xmacex/window-crowparator
```
